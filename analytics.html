Importante: Vercel Analytics ya quedo activado para medir visitas. Para clics y conversiones gratis conviene usar GA4 y Clarity. La web no envia nombres, direcciones ni descripciones del problema como eventos.

Para activarlo completo

  1. Vercel Analytics: ya esta activado. Entra al panel y visita la web una vez para que empiece a contar.
  2. GA4: entra en Flujos de datos, abre el flujo web y copia el ID de medicion que empieza con G-.
  3. Clarity: crea el proyecto, copia el codigo que aparece despues de clarity.ms/tag/ y activa el enmascarado de texto.
  4. Search Console: usa Prefijo de URL con https://24-climatizaciones.vercel.app/, verifica con etiqueta HTML y envia /sitemap.xml.
